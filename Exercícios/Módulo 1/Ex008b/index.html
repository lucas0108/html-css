Texto pequeno, ou letras miúdas

Apesar da tag <big>, que deixa as letras grandes ter caído em desuso, a tag <small>, que deixa as letras miúdas ainda é usada.

Textos sobrescritos e subscritos

Temos também as tags <sup> e <sub> que são, respectivamente, as tags responsáveis em x20+3 pelo 20 e em H2O pelo 2

Trechos de código

Temos também a tag <code>, que deixa o texto assim:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR) else: print(f'O número {num} é ÍMPAR') print('FIM DO CÓDIGO')

Porém notamos que o código ainda não está apresentado da mesma forma que seria num VS Code por exemplo, com as identações e quebras de linha. Para isso utilizamos a tag <pre> em conjunto com o <code> e obtemos isso:


    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR)
    else:
        print(f'O número {num} é ÍMPAR')
    print('FIM DO CÓDIGO')

com as indentações e quebras de linha todas respeitadas.

Citações

Para fazermos citações temos duas tags, o <q>, que coloca aspas para uma frase, e temos o <blockquote>, que cria um paragrafo de citação, por exemplo:

O homem é o lobo do homem

Abreviações

Temos também como explicar uma abreviação com a tag <abbr>, como por exemplo na frase: Estou aprendendo HTML e CSS e estou animado.
